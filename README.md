# Amirhossein Khanlari

PhD student in Computer Science at **Stony Brook University**, advised by [Prof. Amir Rahmati](https://amir.rahmati.com/) in the [Ethos Lab](https://ethos.cs.stonybrook.edu/).

My research focuses on **mobile and iOS security**, **vulnerability research**, and **LLM-assisted security analysis** — using large language models to scale up tasks like SBOM analysis, exploit synthesis, and binary reverse engineering.

---

### Current work

- **Third-party iOS app store ecosystem.** Static and dynamic analysis of 1,500+ iOS app binaries distributed outside the App Store. First-author paper under review at **USENIX Security 2026**.
- **LLM-assisted exploit generation from SBOMs.** Parsing software bills-of-materials, matching against CVE corpora, and prompting LLMs to synthesize proof-of-concept exploits to quantify supply-chain risk.
- **iOS reverse engineering tooling.** Building a combined static/dynamic toolchain on top of Frida, Hopper, and `otool` for large-scale, reproducible analysis of encrypted iOS binaries.

### Interests

AI security · adversarial robustness · vulnerability research · reverse engineering · supply-chain security · the intersection of LLMs and offensive security
